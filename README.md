# Тестовое задание: Arkanoid

## Технические требования
- **Версия Unity**: 6000.2.10f1
- **Платформа**: Мобильные устройства (Android)

## Запуск в редакторе Unity

Для корректной работы игры в редакторе необходимо активировать режим симуляции мобильного устройства. Рекомендуется использовать пресет **Google Pixel 5**, однако поддерживаются и другие конфигурации мобильных устройств.
